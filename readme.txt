== Description ==

This plugin sends a message to the Team Messaging Application into a Group Telling the 

Group that a new PRM Access Request form has been submitted.

= Features =

<ul>
<li>RingCentral Embedded Phone app - 
RingCentral's embedded phone app can be turned on or off and calls can be made from within the WordPress Admin area.
</li>

</ul>

== Installation ==

1. Use WordPress' Add New Plugin feature, Upload the ZIP file
2. Activate the plugin 
3. There is no menu item for this plugin, it will only appear on the plugin 
list as active, inactive, etc.
